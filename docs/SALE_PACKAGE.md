## Technical Differentiators

**1. Browser-Native DAW**
Full multi-track DAW running entirely in the browser via Web Audio API. No plugins, no downloads.

**2. AI Mixing Layer**
Python-based AI mixing engine (`ai_mix.py`) with an agent orchestration system. Real-time mix analysis, automated mastering, intelligent FX recommendations.

**3. VST-Style Plugin System**
A browser-native VST architecture allowing modular instrument and effects plugins — rare in web-based audio platforms.

**4. IndexedDB Persistence**
Client-side project caching via IndexedDB for offline-capable project management.

**5. MIDI Hardware Support**
Full WebMIDI API integration — any hardware controller works out of the box.

**6. Mobile-Friendly**
Touch-optimized UI with multi-touch support for drum pads and piano.

**AI/Audio:** Python AI service, librosa (audio analysis), standardized-audio-context, Tone.js, audio worklets, lamejs (MP3 encoding), audiobuffer-to-wav
